# Design de controle

Projeto das funções de controle e da malha PID a serem prontificadas em código.

## Escopo do alinhamento

- Estrutura da malha PID
- Funções de controle a modelar em código
- Definição clara de inputs e outputs
- Critérios de desempenho e segurança

## Registros

| Elemento de design | Status | Observação |
| ------------------ | ------ | ---------- |
| Malha PID | Em definição | — |
| Funções de controle | Em definição | Articular com Software |

!!! warning "Segurança"
    Alterações de ganhos ou limites devem registrar risco, envelope seguro e procedimento de teste.

!!! note "Encaminhamento para Software"
    Após a modelagem, as funções devem ser unificadas e implementadas no repositório pela área de Software.
